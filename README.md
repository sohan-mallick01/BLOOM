🌸 Project BLOOM
A Hinglish Emotional Support Bot & Positivity Hub for Indian Youth
BLOOM is a mobile-first Progressive Web App (PWA) built to tackle the silent mental health crisis among Indian students. It bridges the gap between emotional distress and professional help using a relatable, "Desi" context.


🚀 Key Features
💬 Hinglish Intent Engine: A custom-built scoring logic that understands code-mixed Hindi and English emotions (e.g., "Tension ho rahi hai").

🎬 Bloom TV: A curated feed of motivational content via YouTube IFrame API to divert panic.

📱 PWA Technology: Installable on any phone without an app store; works on low data.

📞 One-Tap Crisis Support: Immediate connection to National Helplines like Tele-MANAS (14416).

🔒 Privacy-by-Anonymity: No login required. All chat history stays on the user's device (Local Storage).
